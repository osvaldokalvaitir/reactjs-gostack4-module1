## :wrench: Install and run

Open terminal:

```sh
# Clone this repo
git clone https://github.com/osvaldokalvaitir/reactjs-gostack4-module1

# Entry in folder
cd reactjs-gostack4-module1

# Install deps with npm or yarn
npm install | yarn

# Launch the app with npm or yarn
npm dev | yarn dev
```
